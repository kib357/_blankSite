Любая структура данных

Неограниченное количество сущностей, множество типов данных и вариантов отображения. Любые типы связи данных: 1-1, 1-N, M-N

4 варианта отображения данных

Таблицы, списки, карточки элементов, одиночный элемент — можно настроить любой вид для каждого хранилища (сущности). Настраиваемая зона предпросмотра.

Фильтрация и сортировка

Фильтры по любым условиям настраиваются за несколько минут. Сортировка по предустановленныму полю в списке, мгновенная по любому полю в таблицах

Скорость и безопасность

Гибкая система разграничения прав доступа защитит ваши данные, а встроенный движок полнотекстового поиска сделает работу с ними быстрой и комфортной.

Встроенные телефония и почта

Мы уже подключили IP телефонию и средства для работы с электронной почтой. Просто введите настройки для ваших поставщиков услуг связи, и пользуйтесь.
