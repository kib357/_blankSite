Blank разработан на современных и популярных языках программирования — Go и JavaScript.
Клиентская часть — SPA на платформе React JS. Данные в приложении обновляются в режиме реального времени благодаря двунаправленному каналу связи с сервером по протоколу WebSocket.
При разработке на Blank не нужно изучать Go — все делается на JavaScript, для этого в сервер приложения встроен интерпретатор сценариев, поддерживающий стандарт ECMAscript 5.1.
